# Jászai Mari Színház – weboldal

Ez a weboldal a tatabányai Jászai Mari Színházat mutatja be.
A projekt iskolai csapatmunkában készült.

# Az oldal tartalma
- Főoldal
- A színház története
- Előadások
- Színészek
- Kapcsolat
## Felhasznált technológiák
- HTML
- CSS
- GitHub (verziókezelés)
- ClickUp (feladatkezelés)

# Csapattagok és feladatok
- *1. tanuló:* Főoldal elkészítése
- *2. tanuló:* A "színházról", az "előadások" oldal elkészítése
- *3. tanuló:* A "színészek", a "kapcsolat" oldal elkészítése
- *4. tanuló:* A css elkészítése
